# IA Min-Max Yalta — Plan d'implémentation

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implémenter une IA min-max multi-threadée (approche paranoïaque à 3 joueurs) qui peut remplacer 1, 2 ou 3 joueurs humains, conformément aux exigences du PDF projet.

**Architecture:** L'IA repose sur une copie profonde de `Partie` pour explorer l'arbre de jeu sans modifier l'état réel. Au niveau racine, chaque coup candidat est évalué dans un thread séparé (`std::async`). L'approche "paranoid" traite le joueur IA comme maximisant et tous les adversaires comme minimisant — extension directe du min-max 2-joueurs décrit dans l'annexe PDF.

**Tech Stack:** C++20, `std::async` / `std::future`, Qt6 (`QTimer`, `QCheckBox`, `QSpinBox`), GoogleTest

- [ ] **Step 4 : Implémenter copy constructor + `operator=` dans `Partie.cpp`**

```cpp
Partie::Partie(const Partie& other)
    : joueurs_(other.joueurs_)
    , historique_(other.historique_)
    , tourCourant_(other.tourCourant_)
{
    for (const auto& p : other.pieces_)
        pieces_.push_back(p->clone());
    for (const auto& p : pieces_)
        plateau_.poser(p->getPosition(), p.get());
}

Partie& Partie::operator=(const Partie& other) {
    if (this == &other) return *this;
    // Réinitialiser plateau
    for (const auto& [pos, _] : plateau_.cases())
        const_cast<Plateau&>(plateau_).retirer(pos);
    pieces_.clear();

    joueurs_     = other.joueurs_;
    historique_  = other.historique_;
    tourCourant_ = other.tourCourant_;

    for (const auto& p : other.pieces_)
        pieces_.push_back(p->clone());
    for (const auto& p : pieces_)
        plateau_.poser(p->getPosition(), p.get());
    return *this;
}
```

> **Note :** `Plateau::cases()` retourne `const unordered_map<Position, Piece*>&`. Pour itérer dans `operator=`, on récupère les positions avant de vider.

Alternative plus simple pour `operator=` : utiliser copy-and-swap.

```cpp
Partie& Partie::operator=(const Partie& other) {
    Partie tmp(other);
    std::swap(plateau_,     tmp.plateau_);
    std::swap(pieces_,      tmp.pieces_);
    std::swap(joueurs_,     tmp.joueurs_);
    std::swap(historique_,  tmp.historique_);
    std::swap(tourCourant_, tmp.tourCourant_);
    return *this;
}
```

**Utiliser la version copy-and-swap ci-dessus** (plus sûre).

> **Prérequis :** `Plateau` doit être swappable. Vérifier que `Plateau` n'a pas de copy constructor supprimé. Si nécessaire, ajouter `Plateau(const Plateau&) = default;` et `Plateau& operator=(const Plateau&) = default;` dans `Plateau.h` (les raw pointers se copient correctement — c'est l'intention ici).

```cpp
std::vector<Coup> Partie::coupsLegaux() const {
    Couleur c = joueurActif().couleur;
    std::vector<Coup> legaux;
    for (const auto& p : pieces_) {
        if (p->getCouleur() != c) continue;
        for (Position dest : p->coupsPossibles(plateau_)) {
            TypeCoup t = TypeCoup::NORMAL;
            // Promotion : pion qui atteint le rang 4 (dernière rangée du secteur)
            if (p->getType() == TypePiece::PION && dest.rang == 4)
                t = TypeCoup::PROMOTION;
            Coup coup{p->getPosition(), dest, t, TypePiece::REINE};
            Partie copie = *this;
            if (copie.jouerCoup(coup))
                legaux.push_back(coup);
        }
    }
    return legaux;
}
```

```cpp
#pragma once

#include "modele/Coup.h"
#include "modele/Couleur.h"
#include "modele/Partie.h"

namespace yalta {

/**
 * @brief IA utilisant l'algorithme min-max avec parallélisation niveau 1.
 *
 * Approche "paranoïaque" pour 3 joueurs : le joueur IA maximise son score,
 * les adversaires (humains ou autres IA) le minimisent chacun à leur tour.
 * Au niveau racine, chaque coup candidat est évalué dans un thread séparé.
 */
class IAMinMax {
public:
    explicit IAMinMax(int profondeur = 3);

    /// Retourne le meilleur coup pour le joueur actif dans @p partie.
    [[nodiscard]] Coup meilleurCoup(const Partie& partie) const;

private:
    int profondeur_;

    /// Valeur heuristique de @p partie du point de vue de @p joueurIA.
    static int evaluer(const Partie& partie, Couleur joueurIA);

    /// Exploration récursive min-max (séquentielle, appelée par meilleurCoup).
    int minmax(Partie partie, int profondeur, bool maximisant, Couleur joueurIA) const;
};

} // namespace yalta
```

```cpp
#include "IAMinMax.h"
#include "modele/TypePiece.h"

#include <algorithm>
#include <limits>

namespace yalta {

namespace {

int valeurPiece(TypePiece t) {
    switch (t) {
        case TypePiece::PION:     return 10;
        case TypePiece::CAVALIER: return 30;
        case TypePiece::FOU:      return 30;
        case TypePiece::TOUR:     return 50;
        case TypePiece::REINE:    return 90;
        case TypePiece::ROI:      return 10000;
    }
    return 0;
}

} // namespace

IAMinMax::IAMinMax(int profondeur) : profondeur_(profondeur) {}

int IAMinMax::evaluer(const Partie& partie, Couleur joueurIA) {
    int score = 0;
    for (const auto& [pos, piece] : partie.plateau().cases()) {
        if (!piece) continue;
        int v = valeurPiece(piece->getType());
        score += (piece->getCouleur() == joueurIA) ? v : -v;
    }
    return score;
}

int IAMinMax::minmax(Partie partie, int profondeur, bool maximisant, Couleur joueurIA) const {
    if (profondeur == 0 || partie.estTerminee())
        return evaluer(partie, joueurIA);

    auto coups = partie.coupsLegaux();
    if (coups.empty())
        return evaluer(partie, joueurIA);

    if (maximisant) {
        int meilleur = std::numeric_limits<int>::min();
        for (const auto& coup : coups) {
            Partie copie = partie;
            copie.jouerCoup(coup);
            bool prochainMaximise = (copie.joueurActif().couleur == joueurIA);
            int val = minmax(std::move(copie), profondeur - 1, prochainMaximise, joueurIA);
            meilleur = std::max(meilleur, val);
        }
        return meilleur;
    } else {
        int meilleur = std::numeric_limits<int>::max();
        for (const auto& coup : coups) {
            Partie copie = partie;
            copie.jouerCoup(coup);
            bool prochainMaximise = (copie.joueurActif().couleur == joueurIA);
            int val = minmax(std::move(copie), profondeur - 1, prochainMaximise, joueurIA);
            meilleur = std::min(meilleur, val);
        }
        return meilleur;
    }
}

Coup IAMinMax::meilleurCoup(const Partie& partie) const {
    Couleur joueurIA = partie.joueurActif().couleur;
    auto coups = partie.coupsLegaux();
    if (coups.empty()) return {};

    Coup meilleur = coups.front();
    int  meilleureValeur = std::numeric_limits<int>::min();

    for (const auto& coup : coups) {
        Partie copie = partie;
        copie.jouerCoup(coup);
        bool prochainMaximise = (copie.joueurActif().couleur == joueurIA);
        int val = minmax(std::move(copie), profondeur_ - 1, prochainMaximise, joueurIA);
        if (val > meilleureValeur) {
            meilleureValeur = val;
            meilleur = coup;
        }
    }
    return meilleur;
}

} // namespace yalta
```

```cpp
Coup IAMinMax::meilleurCoup(const Partie& partie) const {
    Couleur joueurIA = partie.joueurActif().couleur;
    auto coups = partie.coupsLegaux();
    if (coups.empty()) return {};

    // Lancer un thread par coup de premier niveau
    std::vector<std::future<int>> futures;
    futures.reserve(coups.size());

    for (const auto& coup : coups) {
        futures.push_back(std::async(std::launch::async, [&, coup]() {
            Partie copie = partie;
            copie.jouerCoup(coup);
            bool prochainMaximise = (copie.joueurActif().couleur == joueurIA);
            return minmax(std::move(copie), profondeur_ - 1, prochainMaximise, joueurIA);
        }));
    }

    Coup meilleur = coups.front();
    int  meilleureValeur = std::numeric_limits<int>::min();

    for (std::size_t i = 0; i < coups.size(); ++i) {
        int val = futures[i].get();
        if (val > meilleureValeur) {
            meilleureValeur = val;
            meilleur = coups[i];
        }
    }
    return meilleur;
}
```

```cpp
void ControleurJeu::setJoueurIA(Couleur c, bool estIA) {

    else        joueursIA_.erase(c);
}

void ControleurJeu::setProfondeurIA(int profondeur) {
    profondeurIA_ = std::clamp(profondeur, 1, 5);
}

QString ControleurJeu::messageTour() const {
    const Joueur& j = partie_.joueurActif();
    QString nom;
    switch (j.couleur) {
        case Couleur::BLANC: nom = "Blanc"; break;
        case Couleur::NOIR:  nom = "Noir";  break;
        case Couleur::ROUGE: nom = "Rouge"; break;
    }
    bool estIA = joueursIA_.count(j.couleur) > 0;
    return QString("Tour : %1%2").arg(nom, estIA ? " (IA)" : "");
}

void ControleurJeu::onCaseCliquee(Position pos) {
    // Ignorer les clics quand c'est le tour d'une IA
    if (joueursIA_.count(partie_.joueurActif().couleur)) return;

    if (!selection_) {
        // Premier clic : sélectionner la pièce
        auto* piece = partie_.plateau().pieceEn(pos);
        if (!piece || piece->getCouleur() != partie_.joueurActif().couleur) return;
        selection_ = pos;
        auto coups = partie_.coupsLegaux();
        std::vector<Position> dests;
        for (const auto& c : coups)
            if (c.origine == pos) dests.push_back(c.destination);
        vue_.surbrillance(dests);
    } else {
        // Deuxième clic : tenter le coup
        Coup coup{*selection_, pos};
        // Détection promotion
        auto* piece = partie_.plateau().pieceEn(*selection_);
        if (piece && piece->getType() == TypePiece::PION && pos.rang == 4)
            coup.type = TypeCoup::PROMOTION;
        selection_.reset();
        vue_.surbrillance({});

        if (partie_.jouerCoup(coup)) {
            vue_.rafraichir();
            emit tourChange(messageTour());
            // Déclencher l'IA si le prochain joueur est une IA
            if (!partie_.estTerminee() && joueursIA_.count(partie_.joueurActif().couleur))
                QTimer::singleShot(100, this, &ControleurJeu::jouerTourIA);
        }
    }
}

void ControleurJeu::jouerTourIA() {
    // Joue tous les tours IA consécutifs
    while (!partie_.estTerminee()) {
        Couleur actuel = partie_.joueurActif().couleur;
        if (joueursIA_.count(actuel) == 0) break;

        IAMinMax ia(profondeurIA_);
        Coup coup = ia.meilleurCoup(partie_);
        if (!partie_.jouerCoup(coup)) break;

        vue_.rafraichir();
        emit tourChange(messageTour());
    }
}
```
